Sobre a aplicação

Aplicação para cadastro de clientes e ordem de serviço para montagem de móveis.

Detalhes do Projeto

Versão: 1.0

Licença: MIT

Autor: Erica Viana

Descrição: Sistema desenvolvido para ajudar no gerenciamento de dados de clientes e cadastro de ordem de serviço.

Funcionalidades

  • Cadastro de clientes com campos essenciais como nome, endereço e contato.
  • Cadastro de ordem de serviço para montagem de móveis
  • Interface simples e intuitiva.
  • Armazenamento local dos dados (offline).
